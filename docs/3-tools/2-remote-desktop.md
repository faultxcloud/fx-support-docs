# Remote Desktop
Sebuah software akses jarak jauh (remote desktop) sumber terbuka (open-source) yang memungkinkan Anda mengontrol komputer atau perangkat lain melalui internet.

RustDesk dirancang sebagai alternatif gratis dan aman untuk aplikasi remote desktop komersial populer seperti TeamViewer dan AnyDesk.

RustDesk dapat berjalan di berbagai sistem operasi, termasuk:
- Microsoft Windows
- Apple macOS
- Linux (termasuk distro umum seperti Ubuntu, Fedora, dll.)
- Android
- Apple iOS/iPadOS
- Web (akses melalui browser)
